Customer Order Volume Forecast Heatmap Project

Project Overview

This project includes various heatmaps and data analyses to visualize and forecast customer order volume across different regions and product types. It aims to provide insights into customer behaviors and help optimize decision-making in logistics and sales forecasting.

Key Components

Heatmaps

  • Customer Order Density by Count
  • Customer Order Density by Weighted Product Revenue
  • Customer Order Density with Installation Resource Markers

Python Scripts

  • Add Longitude and Latitude to Customer Orders
  • Generate Customer Order Workbook
  • Generate Heatmap by Count
  • Generate Heatmap by Weighted Product Revenue
  • Generate Heatmap with Installation Resource Markers

Data Files

  • Customer Orders Data
  • Longitude and Latitude Data
